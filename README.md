# KpChatGpt 


- KpChatGpt项目结构

```
├── cache      //缓存 
├── configs    //配置文件
├── controller //控制层
├── daos       //dao层，数据持久层
├── docs       //文档
├── e          //error
├── handle     //请求相关
├── logs       //日志
├── middleware //中间件
├── model      //数据库model
├── routers    //路由
├── services   //服务层(具体业务都在这里展开)
├── test   //测试
├── util       //工具包
├── main.go    //主程序
├── makefile   
└── readme.md
```
